# Documentation

## Directories
- `architecture/adr/` -- Architecture Decision Records (numbered, immutable once accepted)
- `architecture/` -- C4 model diagrams, data flow, network topology
- `tutorial/` -- Phase 0-10 tutorial documents matching `.claude/commands/`
- `runbooks/` -- Operational procedures for incidents, scaling, cost management
- `api/` -- GraphQL schema documentation

## Conventions
- ADRs use the format: title, status, context, decision, consequences
- Tutorial phases correspond 1:1 with Claude Code slash commands
- Runbooks are step-by-step procedures, not reference docs
