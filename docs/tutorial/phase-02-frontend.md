# Phase 2: Frontend

## Summary

Create the Next.js frontend with Tailwind CSS and Shadcn/ui components. This phase builds the chat interface, document upload panel, and bot selector -- all connected to the GraphQL API via generated typed hooks.

## Learning Objectives

- Initialize a Next.js 14+ app with the App Router
- Configure Tailwind CSS and integrate Shadcn/ui components
- Generate TypeScript GraphQL hooks with codegen
- Build responsive chat UI with streaming message display

## Key Commands

```bash
# Start the frontend dev server
npm run dev -- --port 3000

# Generate GraphQL types from the backend schema
npm run codegen

# Run component tests
npm run test
```

## Slash Command

Run `/02-frontend` in Claude Code to begin this phase.
